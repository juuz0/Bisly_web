Extras/Notes

  • For "ban and kick" commands, you should provide the user and then the reason.
  • For "createchannel" command, you should provide first the channel name, and then the channel type.
  • For "deletechannel" command, you should either provide the id of the channel, or you will have to mention the channel.
  • For "report" command, you should first make a channel named "reports" in the server.
  • Note that, "report" command can be used by anyone in the server.
  • In "feedback" command, please provide a clear feedback so that the developer can easily understand and work on it.
  • In bulkyt and yt (both are youtube commands) provide sufficient arguments to get accurate results.
